Project 1.1.9 - Grow Your Own Garden Program

Our 1.1.9 project features multiple bugs, a watering can, and flowers to illustrate our knowledge of turtle. The spider, ladybug, and watering can are stationary.

1.2.5 Project - Spaceship!

This is a game where a user controls a space ship to gather all the coins. The game ends when 10 coins have been collected. The user uses keys WASD to control the ship
and when the ship reaches a coin it dissapears and randomly appears elsewhere.

Project 1.3.1 - BattleFlight

This is a game where a user controls a plane which can move using WASD, and shoot bullets to destroy enemy planes using the spacebar which allows the user to shoot at the
enemies. There is alos a leaderboard to check your ranking amongst those who have played the game. The enemies also randomly appear and move until they are shot down.
The game ends when your plane crashes into an enemy plane.

Project Scratch - Bubble Clicker!

This is a game where a user clicks bubbles while a fish keeps track of your score, score is determined by the size of the clicked bubble(the smaller the bubble the
higher score you get). There are multiple backgrounds which can be cycled through using the spacebar. The bubble changes appearance everytime it is clicked and
moves to a random location. There is also a for inputting the user's name which is later used to display the score alongside the users name.
